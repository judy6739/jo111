أساسيات جافا سكريبت

مقدمة

جافا سكريبت (JavaScript) هي لغة برمجة ديناميكية تُستخدم في تطوير الويب لجعل الصفحات التفاعلية والممتعة. تُعتبر جافا سكريبت واحدة من ثلاث تقنيات أساسية لتطوير الويب، إلى جانب HTML وCSS. في هذا المقال، سنتناول أساسيات جافا سكريبت ونتعرف على كيفية استخدامها.

1. ما هي جافا سكريبت؟

جافا سكريبت هي لغة برمجة تستخدم بشكل أساسي لإنشاء محتوى ديناميكي على صفحات الويب. تُستخدم لتفعيل التفاعلات على المواقع، مثل استجابة الأزرار، والتعديل على المحتوى، وإجراء العمليات الحسابية، وغيرها من المهام.

2. تضمين جافا سكريبت في صفحة HTML

يمكن تضمين كود جافا سكريبت في صفحة HTML باستخدام وسم <script>. يمكن أن يكون الكود داخل الصفحة نفسها أو في ملف خارجي. إليك مثالًا على كلا الطريقتين:

تضمين كود جافا سكريبت داخل الصفحة:

<!DOCTYPE html>
<html>
<head>
    <title>مثال على جافا سكريبت</title>
</head>
<body>
    <h1>مرحبًا بكم في جافا سكريبت!</h1>
    <button onclick="displayMessage()">اضغط هنا</button>

    <script>
        function displayMessage() {
            alert("أهلاً وسهلاً بك في جافا سكريبت!");
        }
    </script>
</body>
</html>

تضمين كود جافا سكريبت من ملف خارجي:

<!DOCTYPE html>
<html>
<head>
    <title>مثال على جافا سكريبت</title>
    <script src="script.js"></script>
</head>
<body>
    <h1>مرحبًا بكم في جافا سكريبت!</h1>
    <button onclick="displayMessage()">اضغط هنا</button>
</body>
</html>

محتوى الملف script.js سيكون كما يلي:

function displayMessage() {
    alert("أهلاً وسهلاً بك في جافا سكريبت!");
}

3. المتغيرات

يمكنك تخزين البيانات في المتغيرات باستخدام الكلمة المفتاحية let أو const. إليك مثالًا:

let name = "جودي";
const age = 14;

4. الأنواع الأساسية للبيانات

جافا سكريبت تدعم أنواع البيانات الأساسية التالية:

  • String: سلسلة من الأحرف، مثل "Hello".
  • Number: الأرقام، مثل 42 أو 3.14.
  • Boolean: قيمتين، true أو false.
  • Object: مجموعة من القيم المرتبطة.

5. العبارات الشرطية

تُستخدم العبارات الشرطية مثل if وelse لتنفيذ كود بناءً على شروط معينة:

let score = 85;

if (score >= 60) {
    console.log("لقد نجحت!");
} else {
    console.log("حاول مرة أخرى.");
}

6. الحلقات

تُستخدم الحلقات لتكرار تنفيذ كود معين. هناك عدة أنواع من الحلقات، مثل for وwhile:

for (let i = 0; i < 5; i++) {
    console.log("الدورة رقم: " + i);
}

7. الدوال

تعتبر الدوال كتل من الكود يمكن إعادة استخدامها. إليك كيفية تعريف دالة:

function greet(name) {
    return "مرحبًا، " + name + "!";
}

console.log(greet("جودي"));

8. التعامل مع الأحداث

يمكنك استخدام جافا سكريبت للتفاعل مع أحداث المستخدم مثل النقر على زر أو كتابة نص:

<button id="myButton">اضغط هنا</button>

<script>
    document.getElementById("myButton").addEventListener("click", function() {
        alert("تم الضغط على الزر!");
    });
</script>

الخاتمة

تُعتبر جافا سكريبت لغة برمجة قوية ومرنة. من خلال فهم الأساسيات التي تم تناولها في هذا المقال، يمكنك البدء في بناء تطبيقات ويب ديناميكية وتفاعلية. استمر في التعلم والتجريب لتحسين مهاراتك في جافا سكريبت!
